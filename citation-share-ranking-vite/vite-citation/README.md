# Citation Share Ranking — Vite Starter

Standalone Vite + React + TypeScript + Tailwind project containing the
`CitationShareRanking` component.

## Getting started

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Files

- `index.html` — Vite entry HTML
- `src/main.tsx` — React entry, mounts the component
- `src/CitationShareRanking.tsx` — the component
- `src/index.css` — Tailwind + design tokens (HSL CSS vars)
- `tailwind.config.js` — semantic color tokens mapped to the CSS vars
- `vite.config.ts` — Vite + React plugin
